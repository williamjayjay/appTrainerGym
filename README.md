## :information_source: How To Use

1- Clone the repository

```bash
git clone https://github.com/williamjayjay/appTrainerGym
```

2- Install the dependencies
```bash
yarn
```

3- Run web application
```bash
yarn start
```

OR 
```bash
yarn android 
```

OR 
```bash
yarn ios 
```
